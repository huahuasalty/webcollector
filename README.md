# webcollector
webcollector+selenium demo


this is a small demo for testing webcollector+selenium

java 爬虫：
webcollector以及webmagic 简单应用
使用selenium 来动态获取js.
 
webmagic中使用selenium 以及模拟表单post请求获取下一页数据

webcollector中通过自定义Requeseter来设置请求头信息，解决乱码问题

USE MAVEN

        </dependency>
